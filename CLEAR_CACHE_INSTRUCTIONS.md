# Clear Browser Cache - Manual Instructions

If the "Clear Cache & Update" button in the app doesn't work as expected, you can manually clear your browser cache using these methods:

## Method 1: Hard Reload (Quickest)

### Windows/Linux:
- **Chrome/Edge**: Press `Ctrl + Shift + R` or `Ctrl + F5`
- **Firefox**: Press `Ctrl + Shift + R` or `Ctrl + F5`

### Mac:
- **Chrome/Edge**: Press `Cmd + Shift + R`
- **Firefox**: Press `Cmd + Shift + R`
- **Safari**: Press `Cmd + Option + R`

## Still Having Issues?

If cache clearing doesn't resolve your issue:

1. **Try Incognito/Private Mode**:
   - Chrome/Edge: `Ctrl + Shift + N`
   - Firefox: `Ctrl + Shift + P`
   - Safari: `Cmd + Shift + N`

2. **Disable Browser Extensions**: Some extensions may interfere with the app

3. **Update Your Browser**: Make sure you're using the latest version

4. **Try a Different Browser**: Test if the issue persists in another browser
